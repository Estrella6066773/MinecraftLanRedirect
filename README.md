# MinecraftLanRedirect

一个简单的小工具，**将远程Minecraft服务器伪装成局域网房间**，让局域网内的玩家可以像加入本地世界一样一键连接远程服务器。

## ✨ 核心功能

- **端口转发**：将本地端口接收的连接转发到远程服务器
- **LAN伪装**：在局域网中广播假的服务器信息，让客户端在"局域网世界"列表中看到
- **安全控制**：支持IP白名单，保护你的服务器
- **简单配置**：通过YAML配置文件轻松设置所有参数

## 🚀 快速开始

如果你对以下内容感到不解，请自行学习**控制台**和**批处理文件**的相关知识。

### 1. 下载或构建
```bash
# 使用Maven构建
mvn clean package
```
构建后会在 `target` 目录生成可执行Jar文件。

### 2. 首次运行生成配置
```bash
# 注意：下载的软件包有版本号后缀，请自行重命名
@echo off
chcp 65001 # 如果中文出现乱码，请添加此行
java -jar MinecraftLanRedirect.jar
```
程序会自动在Jar同目录生成 `config.yaml` 配置文件。

### 3. 编辑配置文件
打开生成的 `config.yaml`，主要修改以下部分：
```yaml
remote:
  host: "mc.example.com"       # 远程服务器地址
  port: 25565                  # 远程服务器端口

local:
  listenPort: 25570            # 本地监听端口

lan:
  motd: "&a我的远程服务器"      # 在局域网列表中显示的名称
```

### 4. 再次运行
```bash
java -jar MinecraftLanRedirect.jar
```

## 📋 使用场景

- **家庭网络**：让家人方便加入你的远程服务器
- **校园网/公司内网**：在局域网内共享远程服务器
- **网吧环境**：让多台电脑一键连接同一服务器
- **需要IP限制**：只允许特定IP段的玩家连接


## 🔒 安全提示

- 默认情况下允许所有局域网客户端连接
- 如需限制访问，请在 `security.whitelist` 中添加IP段，如：`["192.168.1.0/24"]` (未实现)
- 如果远程服务器需要身份验证，可配置 `credentials` 部分 (未实现)

## 📝 日志查看

程序会输出详细的运行日志，包括：
- 启动信息
- 连接转发状态
- LAN 广播状态
- 安全警告

可通过修改 `logging.level` 调整日志详细程度。
